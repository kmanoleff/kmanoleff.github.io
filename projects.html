Unity - 2D Side Scroller

This is a 2D side scrolling game that encorporates the aspects of the Unity physics engine. Environment and backgrounds use free assets from the Unity Asset Store.

Game Encorporates:

  • RayCasts to detect collisions
  • Collectible objects
  • Moveable platforms
  • End-game scenarios and restarts
  • "Through" platforms that detect only RayCasts below
